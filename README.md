# libjsonrpc

This framework provides TCP/JSON-RPC (remote procedure call) support for C++.
Base on libevent2.1.8

## Required Dependencies
* [glog]
* [gflags]
* [hiredis]
* [libmysqlclient]
* [Libevent2.1.8]
* [CMake]

## Testing Environment
* CentOS Linux 7.0
* Gcc Version 4.8.5
